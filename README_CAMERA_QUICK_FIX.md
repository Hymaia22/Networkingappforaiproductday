# 🚨 Problème "Permission denied" - SOLUTION RAPIDE

## ⚡ Solution Immédiate (10 secondes)

### Si vous voyez l'erreur "Permission denied" :

1. **L'app bascule automatiquement en mode saisie manuelle** ✅
2. **Tapez le code du badge** dans le champ (ex: `6623809483`)
3. **Cliquez sur "Valider"** → Vous êtes connecté ! 🎉

---

## 🔧 Pour Activer la Caméra

### Étape 1 : Vérifier HTTPS

**La caméra ne fonctionne QUE sur HTTPS !**

✅ **Fonctionne :**
- `https://votresite.com`
- `http://localhost:5173` (exception locale)
- `http://127.0.0.1:5173` (exception locale)

❌ **Ne fonctionne PAS :**
- `http://votresite.com`
- `http://192.168.x.x`

### Étape 2 : Autoriser la Caméra

**Google Chrome / Edge :**
1. Cliquez sur 🔒 à gauche de l'URL
2. Cliquez sur "Paramètres du site"
3. Caméra → "Autoriser"
4. Rechargez (F5)

**Firefox :**
1. Cliquez sur 🔒 à gauche de l'URL
2. Caméra → "Autoriser"
3. Rechargez (F5)

**Safari :**
1. Safari → Préférences → Sites web → Caméra
2. Votre site → "Autoriser"
